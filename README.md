# ChipVid
## Description
ChipVid is a stealer from a moodle platform private videos uploaded in vimeo.

# Requirements
## Install libraries
```
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Run
```commandline
python stealer.py
```
